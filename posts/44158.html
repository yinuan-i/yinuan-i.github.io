Python 的安装

发表于 2020-09-01 更新于 2020-09-04 分类于 Python

Python的安装：

进入Python官方网站下载安装包

​ 个人癖好，下载东西就喜欢官网的；应为是外网所以得 科学上网

点击导航栏的 Downloads 会自动识别你的windows系统 你会看到 Python3.8.5（32位系统）的安装包

注：Python3以后版本不再向Python2进行兼容，所有如果是为了学习建议直接下载Python3以后的版本

如果你是系统是windows64位或者你想下载其他版本的安装包—— 请点击Downloads > Windows 下载你需要的版本安装包

注：64位版本不可以安装在32位的系统上，但是32位版本可以安装在64位的系统上

2.将Python安装到你的windows操作系统上

我安装的是64位系统的安装包，所以显示为 Install Python3.7.9（64-bit） ，请根据windows系统需求进行安装

这里以 自定义安装 为例

如果没有特殊需求，就全选上。反正也不大，万一用到了呢 点击 Next 下一步

请根据自己的需求修改安装路径 再点击 Install 进行下一步

等待安装中

安装完成！！
